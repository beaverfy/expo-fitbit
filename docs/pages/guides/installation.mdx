Install expo fitbit with `yarn` or `npm`.
<Tabs items={['yarn', 'npm']}>
<Tabs.Tab>
```fish
yarn add @beaverfy/expo-fitbit
```
</Tabs.Tab>
  <Tabs.Tab>
  ```fish
npm install @beaverfy/expo-fitbit --save
```
  </Tabs.Tab>
</Tabs>
