# Login Button
`LoginButton` is a simple component that wraps the child component with a clickable that will open the login page

```tsx
<LoginButton>
    <Text>Login with Fitbit</Text>
</LoginButton>
```

<Callout type="warning">
`LoginButton` does not wrap the child component with a button or text
</Callout>
